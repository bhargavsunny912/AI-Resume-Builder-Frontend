# 🚀 AI Resume Builder

An intelligent, all-in-one **AI-powered Resume Builder** that helps users **create, refine, and download ATS-friendly resumes** in one place — no need to switch between multiple platforms.

---

## 🎯 Motto

> **Build and refine your resume with AI — all in one place.**

Traditionally, users create resumes on one platform and then use another tool to optimize them for ATS.  
This project eliminates that gap by combining **resume creation + AI enhancement** into a **single seamless experience**.

---

## ✨ Features

- 🧠 **AI-Powered Resume Enhancement**
  - Uses Gemini 2.5 Flash API
  - Generates ATS-friendly content

- 📄 **Resume Creation from Scratch**
  - Add personal, education, skills, and experience details

- 🎯 **Level-Based Resume Optimization**
  - Fresher
  - Intermediate
  - Experienced  
  → AI tailors resume based on selected level

- 👁️ **AI Preview**
  - Preview AI-enhanced resume before downloading

- 📝 **Rich Text Editor**
  - CKEditor integration
  - Supports bold, italic, bullet points

- 📥 **Direct PDF Download**
  - Using React2Print

- 🔔 **Notifications**
  - React Toastify integration

- 🎨 **Modern UI**
  - Built using Tailwind CSS

## 🧠 How It Works
User Input → Select Experience Level → Fill Resume Details
↓
Click "AI Preview"
↓
Gemini AI Refines Resume Content
↓
Preview Enhanced Resume
↓
Download as PDF


---

## 📌 Project Workflow

1. Select experience level (Fresher / Intermediate / Experienced)
2. Fill resume details
3. Write summary using rich text editor
4. Click **AI Preview**
5. AI refines the resume
6. Preview enhanced resume
7. Download as PDF
